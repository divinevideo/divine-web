## Locale strategy

Phase 4 rewrote **English only** (`src/lib/i18n/locales/en/*.json`). The
~15 non-EN locales (ar, de, es, fr, id, it, ja, ko, pt, ru, sw, th, tl, tr,
zh, zh-TW) are now **stale vs EN** and need regeneration via translation in
a follow-up pass. They still load fine and carry the older corporate phrasing
— this is intentional so we can ship the EN voice refresh without blocking
on translation. Flag for a future PR: run the translation pipeline from the
updated EN files.
